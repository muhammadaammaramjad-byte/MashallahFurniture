How to Use:

  1. Drag & Drop: Drag image files directly onto the upload zone
  2. Click to Browse: Click the upload zone or "Choose Files" button
  3. Preview: See thumbnails of selected images
  4. Remove: Hover over preview and click trash icon to remove
  5. Upload: Click cloud icon on individual images or "Upload All" button
  6. Compression: Images are automatically compressed for web optimization

Supported Formats:

JPG, PNG, WebP (max 5 files, 5MB each)

Integration Example:

// Initialize uploader
const uploader = new ImageUploader({
    container: '#myUploader',
    maxFiles: 3,
    maxSize: 2 * 1024 * 1024, // 2MB
    onUpload: (url, filename) => {
        console.log('Uploaded:', url, filename);
        // Add to product gallery
        addImageToGallery(url);
    },
    onError: (error) => {
        console.error('Upload error:', error);
    }
});

// Upload all at once
await uploader.uploadAll();
